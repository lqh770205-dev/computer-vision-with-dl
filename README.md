# Computer Vision with Deep Learning

CNN-based computer vision experiments including **ConvNeXt ablation study** and **object detection comparison (FCOS vs RetinaNet)** using PyTorch.

This project explores how architectural design choices affect model performance in both **image classification** and **object detection** tasks.

---

# Project Overview

The project contains two main components:

1. **ConvNeXt Ablation Study**
   - Evaluate how architectural components impact CNN performance.

2. **Object Detection Comparison**
   - Compare anchor-free and anchor-based detectors on aerial imagery.

Technologies used:

- Python
- PyTorch
- Torchvision
- CNN / ConvNeXt
- Object Detection (FCOS, RetinaNet)

---

# Datasets

### PatchCamelyon

Used for image classification experiments in the ablation study.

Contains histopathology images for binary classification.

### NWPU VHR-10

Used for object detection experiments.

Dataset contains aerial images with objects such as:

- airplanes
- ships
- baseball diamonds
- storage tanks

# ConvNeXt Ablation Study

We performed an ablation study on **ConvNeXt-Tiny** to analyze the impact of architectural components.

Experiments include:

- LayerNorm → BatchNorm
- GELU → ReLU
- 7×7 → 3×3 convolution
- Inverted bottleneck → standard bottleneck
- Removing stochastic depth

# Object Detection: FCOS vs RetinaNet

We compare two popular object detectors:

### FCOS
Anchor-free detector that predicts objects directly from feature maps.

### RetinaNet
Anchor-based detector using Feature Pyramid Networks (FPN) and focal loss.
